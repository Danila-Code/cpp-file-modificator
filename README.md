# Модификатор файлов

## Описание
Программа с графическим интерфейсом, модифицирует входные файлы с помощью применения XOR операции с 8 байтовым ключом.   Хранение элементов осуществляется последовательно во внутреннем массиве. Сам вектор хранит только указатель на этот массив данных и количество элементов в нём. При заполнении всего массива происходит перевыделение памяти для нового массива, размер которого в два раза больше предыдущего. При этом все данные копируюся/переносятся в новый массив.

## Технологии:
* C++
* Qt
* Qt Widget
* многопоточность c QThread
* сигналы и слоты

## В модификаторе реализованы:
* отдельный класс Modificator, который производит всю работу с файлами
* графический интерфейс Qt
* проверка валидности заданния папок входных и выходных файлов
* проверка валидности задания 8 байтового ключа
* однократный запуск модификации файлов или с периодичностью, задаваемой таймером
* индикатор прогресса, который показывает объём выполненной работы
* список с результатами работы, который о отображает имена обработанных файлов и их статус
* сообщения об ошибках открытия и чтения файлов

## Особенности модификатора:
* модификация файлов производится в отдельном потоке, что не даёт интерфейсу "подвисать"
* связь между Modificator'ом и главным окном осуществляется с помощью сигналов и слотов
* обновление индикатора прогресса оптимизировано, чтобы не вызывать "торможение интерфейса" (обновление происходит не чаще, чем через 1%)
* при совпадении имён выходных файлов, когла чекбокс "перезаписывать выходной файл" не выбран, к именам файлов добавляются скобки с индексом, например "text(1).txt"
* сохранение параметров модификатора происходит при нажатии кнопки "Модифицировать файлы"
* если запустить модификацию файлов с флагом "запуск по таймеру" кнопкой "Модифицировать файлы", то текст на кнопке изменится на "Остановить таймер", и повторным нажатием можно таймер остановить
* при запуске модификации файлов с флагом "запуск по таймеру", таймер начинает отсчитывать время после завершения предыдущей модификации
* входные файлы задаются с помощью маски

## Сборка проекта
Вариант 1
1. Скачайте файлы из текущего репозитория.
2. Создайте в корневой папке проекта папку "build" для сборки проекта, перейдите в неё в командной строке и запустите cmake:\
	`cmake ../file_modificator/.`
3. Запустите сборку проекта в командной строке:\
	`cmake --build .`
*Проект собран.*

Вариант 2
1. Скачайте файлы из текущего репозитория.
2. Откройте файл `file_modificator/CMakeLists.txt` в Qt Creator и запустите сборку

## Запуск
Чтобы работать с модификатором файлов запустите FileModificator.exe

## Системные требования
Qt 5, Компилятор С++, С++17, CMake 3.16
